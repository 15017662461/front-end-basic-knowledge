# 设计美学

## 第一讲 设计之美的基本问题

### 设计的产生与发展

+ 设计的提出：文艺复兴时期，意大利的乔尔乔·瓦萨里。他在《著名画家、雕塑家和建筑师生平》（1550）中提出“设计的艺术”这一重要理论。用Disegno涵盖，绘画雕塑建筑三项，称Disegno（设计）为这三项艺术的父亲
+ 设计的发展：1. 文艺复兴以来，设计与大艺术、美术一体。2. 20世纪上半叶，设计与大艺术、美术相对（小艺术、工艺等）3. 20世纪下半叶，设计成为独立一门学科
+ 现代设计（设计与技术）：出现两种观点。
+ + ①批判工业机器的罪恶，希望重构传统手工设计（工业革命背景下）。代表人物有普金、拉斯金、莫里斯
  + ②积极建立机器的美学，高度赞扬规模、工业化的工业产品机器技术的创造力。包豪斯学校的设计教育、设计理论。线代设计成为建设现代社会的主力军。代表人物有勒·柯布西耶、瓦尔特·格罗皮乌斯



### 设计的研究

+ 西方最早的设计理论专著：威廉·贺加斯《美的分析》，是把对装饰艺术的研究与对知觉心理学的研究联系起来的第一人
+ 设计的内涵与特征：
+ + ①是创造物的形式
  + ②与特定的物质生产科学技术有密切的关系，设计具有自然科学的客观性特征
  + ③设计贯彻到人们的思想与情感，设计与特定的政治文化有显而易见的关系。设计具有特殊的意识形态色彩



### 美学的产生与发展

+ 美学的产生：被誉为美学之父的鲍姆加登，第一次使用了“Aesthetic"这一次（美是感官的感受），建立了美学这一特殊的哲学学科（感性学）
+ 美学五力：美学以人类心理能力中的“感性”能力为研究对象。包括：
+ + 感官的认知力（审美感受）
  + 情感的认知力（审美情感）
  + 想象的构建力（审美思维）
  + 审美的反思力（审美判断）
  + 感性的表达力（美的创造）



### 设计美学的学科性质

+ 设计学和美学的交叉学科
+ 从审美的角度来考察设计的问题：
+ + 研究设计造物及其美感的发生、发展、性质、特征及其规律
  + 研究艺术要素、技术要素、文化要素对涉及发展的影响，以及对人类感知模式的塑造
  + 研究设计在感官认知、情感认知、审美反思、感性表达等层面，对人类思维、观念、感知模式和生存方式影响的程度和深度

+ 设计与需要，理解设计美的本质

### 设计美的本质

**设计物体现着人需求的满足。人们从设计物中看到自身需求，人类在更新创造物的同时也在更新完善着自身。**

**设计就是将人的内在尺度融入到造物活动中去**

**设计美的本质，是人类对于合乎内在尺度的造物设计的情感判断，是人的内在尺度在人工造物上的感性体现**

#### 人的需要

1. 人的需要的两个特征：层次性和发展性
2. 马斯洛将人的需要划分为三个层次：
   + 第一层次：生命生存的基本需要，包括安全需求和生理需求
   + 第二层次：社会生存的需要，包括社会需求、尊重的需求、自我实现的需求
   + 第三层次：自我超越的需求
3. 举例：鸟图腾，超越了生命生存的基本需要和社会生存的需要，体现出人文精神，是特殊化了的精神需要的满足



#### 人的尺度与设计美的标准

1. 举例：圆，具有感知美、功能美、形式美——使人愉悦——抽象为一种精神的满足、凝结为一种感觉的经验。标志着对象与心理器官或者功能之间的某种协调或关系，将它作为人的内在尺度和指导人创造美的规律。
2. 内在尺度（设计美的标准）：
   + 真：与人的感觉相适应；在实践创造的过程中发现的客体的形式和规律
   + 善：体现社会人的感觉，遵循社会价值尺度
   + 美：人按照美的规律来构造——主体的尺度体现在对象上，对象的本质体现在主体的尺度中。美是人的内在尺度在自由创造中的生动体现——黄金分割。体现出设计是人们决定其物质未来的唯一方式



#### 人类的设计和设计的人类

设计活动与人类自身之间是互动关系，设计活动通过造物改良提升优化人的物质和精神状态，“人化”的物质形态体现出文化的影响力，改变人的生产生活方式，决定性的作用于人类文明发展和延续。设计的最终结果，是不断改变和完善的人性。



### 设计美的形式

#### 设计美的感知要素

+ 造物的形式美与人的感性和感觉密切相关
+ 感觉的心理力——格式塔心理学（完形心理学）
+ + 人的感知结构不仅是生理的还是心理的
  + 人的感觉并非是简单的刺激反应，而且是具有强大的能动力量
+ 心物同形论
+ + 实验：“悲哀表情实验”——心理力与物理力同型
  + 理论：心理的力，与自然界的物理力一样，都服从着同一个基本的组织规律

#### 设计美的形式要素

心物同形的感知规律——视觉感知经验。有一些视觉要素：点线面体、光影、运动、色彩、材质等。设计利用人视觉要素和心理感知力，创造出具有丰富美感的作品。设计美的形式就是人所感知的有意味的形式



#### 设计美的心理建构

视知觉的心理整合力——点的形式美

空间构造力（重叠与变形）——线构成面

平衡力——对称、均衡之美

空间构造力——面构成体



## 第二讲 设计的媒介力量

### 设计与媒介

#### 设计

设计是将人的内在尺度融入造物活动

设计物就是人的精神价值的感性显现

#### 媒介

赫伯特·马歇尔·麦克卢汉在《理解媒介》中：受控于媒介方式所带来的的思想、情感的一体化，才使受众具有存在感和归属感。其对媒介的定义（广义）：人创作的技术文明，所有创造物都是媒介

#### 媒介技术的详细分类

左边为媒介技术，右边为媒介形态

+ 材料媒介——物质形式
+ 符号媒介——图形、符号形式
+ 影像媒介——影像形式
+ 互动媒介——复合媒介形式

其对应的媒介策略与媒介设计为：

+ 全感知形式——建筑设计、产品设计
+ 视觉形式——传达设计
+ 视听形式——传达设计
+ 互动形式——传达设计



#### 人类的设计物（物质生产和精神生产）

设计物的媒介特征：目的性、精神性（非物质）

设计物具备三要素：媒介技术、信息形态、传达策略。这三者构成媒介意向，所以一切设计都是传达设计，一切媒介都发挥着影响心灵的巨大作用



### 建筑设计的媒介力量

#### 建筑的媒介性

+ 建筑是人类精神与情感的“非有机躯壳”，发挥着整饬人心的场域力量

+ 从媒介上来说，建筑是一种需求的寄托、情感的记忆、理性的反思（是审美反思的媒介）；从功能上来说，建筑满足人活动的需要

+ 举例：夏特尔教堂与格里高利圣咏曲——建筑形式本身表现出节奏和韵律；安藤忠雄——光之教堂、水之教堂


  #### 经典性建筑：

+ + 地标建筑与城市意象：表现着城市的精神层面，是城市的标签和城市文化策源地，包括建筑，公共空间，公共艺术
  + 景观设计与城市意象：景观设计协调人建筑与自然的关系，是生态美学的重要媒介
  + 城市规划与城市意象：赋予环境视觉秩序的能力，赋予城市视觉形象，直观传达出城市的文化取向

**不同的建筑形态揭示了人类对自身理想性生存形态的多样化追求**



### 产品设计的媒介力量

#### 历史背景

造物由按需生产，到现代工业革命规模化大批量生产，产生产品设计活动

#### 产品设计

+ 定义：是将人的某种目的或需要转换为一个具体的物理或工具的创造过程
+ 阐释：在人的自然属性和社会属性的认知基础，将原料的形态改变为更具有价值的形态
+ 意义：通过产品回应社会经济环境伦理方面的问题，引导创新更好的生活

#### 产品设计三段式跨越

+ 手工产品设计。举例：女装设计
+ + 产品形象被认同为使用者形象
  + 产品塑造了现代女性的审美观
  + 产品在精神上赋予了女性平等的权利，反映了自由的根本需求
  + 许诺自由是产品设计的终极目标，产品设计致力于图片自然时空的局限，开拓出更广阔的的生存空间
+ 工业产品设计。举例：汽车的设计与发展
+ + 产品扩展了时空，改变着社会的结构
+ 信息产品设计。举例：通信媒体传播工具
+ + 信息产品打破了人们原有的物理时空的感知框架
  + 一个时代所使用的的传播工具的性质，以及它所开创的可能性和带来的变革，是社会真正有价值的讯息
  + 新时代也出现了多元复杂的网络结构，自媒体使得传统社会中心-边缘的社会结构正在变成多中心的网络结构

**综上，可以看出：1. 产品设计是对规范化和秩序的呈现，对人们的思维方式的影响是持久而且强劲的。2. 产品设计本身是一种修辞的艺术。3.产品对人和社会具有反作用。4. 人们更新着产品，新产品积极塑造这新的秩序规范和精神价值。**



#### 与建筑设计的区别

+ 建筑设计：坚固性、耐久性。塑造文化的审美力量
+ 产品设计：永不止息的进步性。产品拓展了时空，改变着社会结构，不断更新人类的生存样貌，影响着人类的发展



### 视觉传达设计（VI）

+ 将不可见性变成可见性。例子：通过一些壁画传递远古时期的循序：一些不可见的含义化为可见的意向
+ 将可见性变成可读性。“梅兰竹菊”将设计物中的现实要素赋予一定的意义，翻译可理解的文本，指导实际活动
+ 建立精神内容与直接体验的联系。符号化意象：意象是视觉形式背后看不见的逻辑
+ 以能指-所指的逻辑系统建立清晰的意义世界。符号可以将精神空间呈现出来，是统一团体分享信念的请打工具
+ 企业视觉识别系统VI--赋予物质以灵魂。获取内部员工认同感归属感，外部受众认知感与认同感
+ 并非是一种形式，而是人们的期待和愿望。医院导向识别系统
+ 赋予事件以丰满的形象，将人们的期待与愿望放入其中
+ 所见即真实的心理建构，视觉设计上的强化使人身临其境，优化用户体验

**利用图像和符号唤起意象经验（前四个），通过体验直抵人心（后四个），对人的行为思想产生影响**

### 视听传达设计

+ 模拟感官的在场：
+ + 镜头：视觉动觉听觉要素，制约视觉传达的内容和范围
  + 蒙太奇：时空转换，用蒙太奇的逻辑左右我们的思路
+ 通过叙事说服，影像是一种欲望的镜像，映照出主体自身的欲望和形象
+ 通过情感和价值的心理认同来说服。励志视频
+ 通过文化身份认同来获得影响力。影响个体和群体进行自我定义



### 互动传达设计

从视觉传达到多感知传达设计：

+ 被信息传达的同时，表达自己的思想，使用产品的同时，影响他人
+ 信息社会极大的改变了人类的认知心理结构和审美方式，也极大地改变了设计本身

**设计物是意识形态的终端，是情感的表达理性的陈述，也是对秩序规范的呈现**









## 第三讲 设计的文化差异 

### 设计的文化语境

#### 分类

+ 物质文化语境
+ + 地域：气候、环境等的不同
  + 人种：体格、智慧等不同
  + 经济方式：生产类型的不同
+ 观念文化语境
+ + 一个地方的生活思维方式，历史文化传统习俗以及价值观念等决定了如何设计和设计怎样的文明方式。包括：政治模式、思维模式、社会结构、行为规范等



#### 举例，东方与西方

+ 东西方文化语境对比：
+ + 自然环境的差异：中国：内陆；西方：海岛-大陆
  + 生存模式的差异：中国：自给自足；西方：贸易交往
+ 观念文化语境：
+ + 轴心时代，德国思想家卡尔·雅思贝尔斯《历史的起源与目标》中提出，公元前600-公元前300年，人类文明精神的重大突破时期
  + + 出现了众多思想先驱：古希腊的苏格拉底、柏拉图、亚里士多德；中国的老子、孔子
    + 创造了元典，对构成人类处境之宇宙的本质产生了一种理性认识，实现了哲学的突破
    + 各个民族的中心价值大体在这一阶段定型，而这些价值对该民族以后的发展起着范畴的作用
  + 东西方传统观念文化语境的对比：
  + + 政治模式上，中国式农业文明，中央集权制，宗法制是为子民；西方是海洋文明，城邦制，民主制，是为公民
    + 社会结构上，中国是圣人-民本思想；西方是哲人-人本思想
    + 行文规范上，中国是伦理之礼，西方是契约之法
    + 思维模式上，中国是观象制历，观象授时，天人合一；西方是数学观测，数学计算，天人对立
    + 文化精神上，中国是情感理性，西方是客观理性。余英时提出，希腊的轴心突破-外向超越型，中国古代的轴心突破-内向超越型



### 物我观差异下的设计美

#### 什么是物我观

人们怎样看待人与自然的关系，如何掌握、利用自然事物固有规律，如何处理人（包括设计者和它的设计服务对象）和自然之间的关系。设计者对自然界、自然环境、自然条件的认识，规定了其全部思维意识的模式，也决定了设计者自身的价值观

#### 东方人的物我观和造物观

+ 物我观：天人合一，物我相通
+ 造物观：传神写意，制物尚象
+ 举例分析：
+ + 以“意”为重的造物设计：器物上，良渚玉琮是权力的象征；青铜器中含有道和礼，通过比类同情，感悟兴会的方式落实到人心，成为社会的行为准则和社会规范
  + 以“神”为重的造物设计：雕塑上：传神飘逸的风格，只求神韵，不求形似，也表现出古人对美好生活的向往
  + 以“心”为重的造物设计：建筑上：亭台楼阁，通透开敞，融于自然，天人相通，也表现出古人的文人雅兴和精神空间
+ 代表著作：《考工记》，第一部在理论上“重意向”“重传神”的设计理念加以总结的经典
+ 东方造物设计：气韵生动，形神兼备，意向美

#### 西方人的物我观和造物观

+ 物我观：主客二分，物我二分。将人和自然看做对立的两方面
+ 造物观：对最高的普遍本质的模仿、再现、反映
+ 雕塑上：也有人体的黄金比例，形式美。以神话题材日常生活题材为主。表情一般是静穆宁静优美的，而不像东方有神韵
+ 城堡建筑，与中国的传统楼亭不同，表现人绝对力量的造物形式，城堡凌驾于自然之上。
+ 代表著作：《建筑十书》建筑家维特鲁威，古希腊神庙之美，用数学对外形比例等进行了详细的记载。达芬奇，人体比例（黄金比例）：建筑美的理想尺度
+ 举例分析：
+ + 以“形”为重的造物设计：器物上，古希腊陶瓶，造型生动纹样写实。技术上有对人体比例的模仿和缩短法等
  + 以“像”为重的造物设计：雕塑上：人体雕塑遵循黄金比例，有人体美和理想的和谐美。雕塑的主要特点“高贵的单纯和静穆的伟大”（温克尔曼），拥有优美的形式崇高的形式。按照事物应该有的或理想的样子，创造、表现本质的形式美
  + 以“物”为重的造物设计：建筑上：西方传统的城堡建筑，是宏伟的、封闭的人造空间；是合理的、规则的可以用比例表示的安排；对秩序 、形式、数目与比例等约束形式规则的把握。
+ 西方造物设计：仿真美，理想美，形式美



### 物用观差异下的设计美

#### 什么是物用观

主体与自然之物，所造之物的关系。人怎么用物，物怎么为人。这里的“人”不是设计者本身，而是设计者的服务对象；“物”不是自然状态的物质本身，而是由物质材料设计和制作成的人造器物

#### 东方的物用观

+ 民本物用观：则被于民，教化于民，开物成务，备物致用
+ + 衣冠设计：衣冠制
  + 食器设计：礼器
  + 建筑设计：礼制。如紫禁城（左右对称，尊卑分明，伦理清明后，秩序井然）、庭院组合（平面布局，单体建筑反复组合）、纪念碑建筑（有补于世）
+ 具有务实美：中国造物设计以满足现实的伦理需求为出发点，反映出泽被天下有补于世的务实美
+ 物之所用，以物化民。从人们切身理性感性的需求出发进行造物，实现对子民的教化。中国造物设计：民本的 “务实”美

#### 西方的物用观

+ 人本物用观：人神并重
+ + 神 本质上是代表人的理性，注重理智
  + 人 代表感性的世俗欲望，注重情感
+ 举例
+ + 雅典卫城。神庙，宙斯谱系的保护神系统，封闭的内部空间，只有“神”的崇高，为“人”设计的空间并不存在；（人的）剧场，人文精神传统
  + 古罗马。纪念碑建筑，“人”的张扬；基督教合法化后，神被推向了人的彼岸，教堂建筑 成为服务于“神”的建筑
  + 教堂建筑。哥特式教堂 服务于“神”的建筑，内部：神秘崇高的强烈情感，花窗：“非人间”的境界；文字复兴时期 回归“人”的尺度，巴洛克风格：不规则的形式起伏的线条和强烈的情感；缺乏古典主义均衡性的作品，背离了文艺复兴艺术精神，洛可可风格：强烈的感性色彩；奢华奇异享乐主义；18世纪中后期新古典主义 理性的回归，建筑特点“高，大，上” ，纵向高耸；结构立面布局+力学结构向上伸展；外向张扬；空间狭而高，追求高度，对人主体精神的发扬
+ 以满足“人”（理性精神感性欲望）为出发点，反映理想美。西方造物设计：人本的 “理想”美



### 方法观差异下的设计美

#### 方法观的定义

人们用来观察事物和处理问题的方式方法，这样的原则普遍适用于造物设计并起指导作用

#### 东方的方法观

+ 方法观：尊物“性”、任“自然”——“无法”之法。（不法，不饰）。即师法自然融于自然，技巧达到“自由”的境界，创造出情景交融的美感
+ 举例：拙政园
+ + 顺自然，无法
  + 幽深曲折、移步换景。更多的偶然性。
+ 融于自然的 意境美（素朴美，天然美）



#### 西方的方法观

+ 方法观：因逻辑、重客观——科学之法。（循法，用法，雕饰）。
+ + 逻辑原则：从事物的普遍联系和永恒运动中把握
  + 客观原则：分析事物自身的原理并利用它进行科学创造
  + 在混乱的自然中通过高度的组织性与技术性，创造一个理性的人工世界
+ 古埃及人、西亚人；阿尔贝蒂——几何园林与庭园
+ 举例：凡尔赛宫：严谨对称的建筑布局
+ + 人为控制的几何布局几何图案，充满科学理性与精确
  + 人工喷泉人工绿植
  + 重要坐标点的雕塑，配置于轴线的起点终点或交点
  + 视野开阔，雄伟庄重，典雅奢华
+ 征服自然的 理性美（雕琢美，人工美）



### 物美观差异下的设计美

#### 物美观的定义

人们在造物实践过程中培养的审美价值观。（是欣赏鉴别和评判美丑的特殊价值标准）。根据不同的物我观、物用观与方法观得出

#### 东方的物美观

+ “法天贵真”为美；随“性”为美。来自“天人合一”的物我观
+ “雅正中和”为美；简淡为美。来自“民本务实”的物用观
+ “天、地、材、工”的遇合为美；气韵为美。来自“无法之法”的方法观
+ 随性的“气韵美

#### 西方的物美观

+ 人物事理为美；造“形”为美。来自“物我二分”的物我观
+ 华丽奢艳为美；绮丽为美。来自“人本”的物用观
+ 人工雕琢为美；人体为美。来自“客观逻辑”的方法观
+ 造型的“外观美“

#### 举例

+ 瓷器。东方：自然天真，中正清雅为美。西方：造型考究，富贵华丽为美
+ 服装。东方：气韵美，流线美与简淡素雅的材质美（神，动，雅）。西方：人体美，几何体块美与色彩华丽的装饰美（形，静，丽）



## 第四讲 技术革命与设计革命

### 技术革命与设计革命

#### 技术与设计的关系

+ 本质上：技术决定了事物的功能、外观和审美的体验（技术决定设计物，从而决定人们的感知方式感知经验）
+ 设计物的组成：外显的形式和内隐的技术
+ + 设计物的形式（艺术）。形式是界限之上的显性表示，改变了人们看待商品的方式
  + 设计物的功能（技术）。技术是界限之下隐藏着的结构机制，决定产品如何发挥应有的作用
+ 技术要素：作为内隐的结构机制（及其对应决定的设计上技术要素）
+ + 材料。决定：设计的逻辑或技术原理
  + 控制技术。决定：物品的程式或特征
  + 能源动机。决定：设计品的感知模式
